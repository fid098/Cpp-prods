# Phase 4 — STL

**C++ concepts covered:** `std::map`, `std::unordered_map`, `std::vector`, `std::set`, iterators, `<algorithm>`, `std::accumulate`, custom comparators, sorting

Phase 4 trades raw data structures for the Standard Library. Every container is used where it earns its keep in a real finance context.

## Projects

| # | Project | Portfolio story | Key concepts |
|---|---------|----------------|--------------|
| 01 | [limit-order-book](01-limit-order-book/) | Price-time priority order book | `map`, `unordered_map`, iterators |
| 02 | [portfolio-analytics-dashboard](02-portfolio-analytics-dashboard/) | Weights, exposure, concentration | `vector`, algorithms, `accumulate` |
| 03 | [etf-holdings-analyzer](03-etf-holdings-analyzer/) | Parse and rank ETF holdings | maps, sets, sorting |
